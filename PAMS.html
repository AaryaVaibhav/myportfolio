Patient Analysis and Monitoring System

Analysis of database and recording of patient information

Project Details:

This application mainly deals with:

  • Patient Records about various types of treatments involved.
  • Research information for researchers for various patient diseases for analysis and study.
  • Auditing module involved maintenance of billing of doctors and patients and other tasks.
  • SOFTWARE REQUIREMENT :
    1. Java (jdk1.6 and above)
    2. HTML and CSS
    3. SQL
    4. Tomcat Server
    5. Image viewer
    6. OS: Windows / Linux
