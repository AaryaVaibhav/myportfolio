Steganography

GUI Based Information Hiding Project

Project Details:

This project/application was built during the final semester of my Master's course. This project provides a GUI, a user friendlier system, where secret information can easily be hidden with in a picture file. It attains all java features. It is platform independent so that it can be used in any OS. Thus secret information can be transferred to the intended user without giving any suspicion to the unintended user.

This project is developed for hiding information in png or bmp file types only. The scope of the project is implementation of steganography tools for hiding information includes any type of information file and image files and the path where the user wants to save Image and extruded file. This also provides the feature to extract the contents of the file after data storage.
